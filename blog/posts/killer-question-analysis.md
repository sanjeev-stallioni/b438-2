È on line per i nostri clienti il nuovo report che analizza le performance delle domande più importanti definite killer nella check list.

Nei nostri questionari possono essere definite delle domande più importanti che noi chiamiamo KILLER, queste domande generano un punteggio negativo quando non si rispetta lo standard.

Abitualmente i nostri clienti utilizzano le Killer Question per dare maggiore attenzione ad alcuni processi ritenuti core.

Le Killer Question sono visibili nei nostri report in prima pagina.

Il pannello web è all'interno della dashboard dei nostri clienti e analizza l'andamento delle singole Killer Question nel tempo.

Il pannello analizza il punteggio aggregato delle domande in modalità multiciclo e per ogni domanda l'andamento.

Serve a riassumere le performance delle domande più importanti in un pannello web di facile fruizione anche per gli area manager.

È accessibile direttamente dal pannello web.
